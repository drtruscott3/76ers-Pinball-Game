# 76ers Pinball Arcade

A Philadelphia 76ers-themed pinball game built in HTML, CSS, and JavaScript.

## How to play

- `A` = left flipper
- `D` = right flipper
- `Space` = charge and launch ball
- `3` balls per game
- Score points by hitting bumpers, targets, and ramps
- Top 10 leaderboard stores high scores in browser storage

## Files

- `index.html` — game page
- `styles.css` — visual theme and layout
- `script.js` — game logic, scoring, sound, and fullscreen support

To host this on GitHub Pages, upload the folder to a repository in your GitHub account and enable Pages from the repository settings.

Then the site will be available from a URL like:

`https://<your-github-username>.github.io/<repository-name>/`

Replace `<your-github-username>` and `<repository-name>` with your actual GitHub account and repo names.
